Helpful function for variables:
.length - returns the size of an array, array length can't be altered
.includes('') - looks to see if an array includes a value
.concat('') - concatenates two strings

To print out a variable in the middle of string text use ${ varialbeName }

const name = `Siri`; `Hello ${ name }!`;



Exponents are done as "2**3" this result comes to 8

Arry destruction:

const [x, y] = [1, 2];



This sets the arrays 'x' and 'y' to the values '1' and '2' respectively

Some useful functions for arrays are:
.join() - joins the elements of an array into a string with elements seperated by ','
.slice(#, #) - returns a sub array starting from one index up to (but not including) the next index
.sort() - sorts alphabetically

Formate for multidimensional arrays

const coordinates = [[1,3],[4,2]];



For functions with that may have an unknown amount of arguments:

function name(...args) { return args }



Function declare with the keyword 'function' are automatically hoisted, meaning they are run before other code.
Callbacks is when a function is used as a parameter to another function.

function sing(song) { console.log(`I'm singing along to ${song}`); } sing('Let It Go')



forEach value in an array

colors.forEach( (color,index) => console.log(`Color at position ${index} is ${color}`) ); << "Color at position 0 is Red" "Color at position 1 is Green" "Color at position 2 is Blue"
