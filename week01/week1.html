Notes

Local storage: localstorage.get or .set("name","value") to have the browser save things. Client has control of the browser, never assume local storage exists. Local storage is visible to users.

Session storage will last until the browser is closed.

When there is nothing inside a section, it will collapse and have no height.

Remember 'placeholder' tag

When interacting with localStorage the keywords 'getItem' and 'setItem' are used. The thing to remember is the value name. When you ask for a value from local storage you need to know it's value name. When setting a value to the local storage you set it's name and the value associated with the name.

Code Examples

#result_display{ background-color: DarkGrey; width: 300px; height: 150px; }

The above code shows how to give dimensions to a section.

function saveStory(){ let storyName = document.getElementById("name_input").value let storyHTML = document.getElementById("story_editor").value localstorage.setItem(storyName, storyHTML) }

The above code saves the entered in story under the name of storyName in the user's local storage.
